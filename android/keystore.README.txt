Android release signing setup (manual steps)

1) Generate a keystore (run on your machine):
   keytool -genkey -v -keystore nookat996-release.keystore -alias nookat996 -keyalg RSA -keysize 2048 -validity 10000

2) Move the keystore to android/ (or a safe path outside the repo) and add this file to .gitignore:
   android/nookat996-release.keystore

3) Create android/key.properties with content (do NOT commit this file):
   storeFile=<absolute_or_relative_path_to_keystore>
   storePassword=<your_store_password>
   keyAlias=nookat996
   keyPassword=<your_key_password>

4) Our Gradle build reads key.properties if present and signs release accordingly; otherwise it uses debug signing.

5) Bump app version in pubspec.yaml (version: x.y.z+code) before Play upload. Current is 1.0.0+1.
